SALFORD
1916-1965
 
13,364


This database of names has been transcribed by Louise Messik from records held at the Greater Manchester Police Museum.

The Museum's curator, Mr Duncan Broady, wrote:-

"The Registers that we recovered from the basement of the old Salford Police station used a sometimes confusing index number system, so we have decided to store them in boxes numbered 1 to 8, with 3 or 4 books per box. Therefore the individual's index number is prefaced by a 1/, 2/, 3/ etc. to indicate the book within that box. We also have a box marked "X" which contains 4 books of "Enemy Aliens" which seemed to follow its own system. Our earliest entries date from 1916 and the latest about 1965.

If you would like us to supply more information on an individual, please quote the box number and the book number as well as the index number so we will be able to find them."
