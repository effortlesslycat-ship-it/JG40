SEARCH THE SIG GIVEN NAMES DATA BASES (GNDBs)

You can access the GNDB of a Special Interest Group (SIG), such as LitvakSig (Lithuania), by visiting that SIG's web site.  In the table below, selecting the SIG-sponsored GNDB you wish to use will take you directly to the GNDB of the SIG's country.  Four SIG-sponsored GNDB sites are currently available.

The SIG GNDB web sites are completely self-contained.  That is, they include localized versions of the options available on this JewishGen GNDB web site, and they also allow you to use the other facilities of the SIG web site without returning to this JewishGen GNDB web site.  Each SIG web site currently in operation also allows you the option of returning to this JewishGen GNDB web site, in order to use the other 14 GNDBs, or to go directly from the current SIG GNDB site to any other available SIG GNDB site, using the same table as below.
