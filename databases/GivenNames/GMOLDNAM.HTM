"GET MESUDAR":  SEARCH OLD NAMES

This data base contains a full set of the Primary and Subsidiary ("Old", mostly Yiddish) names, and the other names NOT to be written in a Get (e.g., diminutives, names of endearment), which appear in the Hilchot Gitin book "Get Mesudar", written by Rabbi Elazar Mintz at the beginning of the 20th century.  You can search for names in any of these three categories.  For the most part, the names in this data base apply to Germany or Poland, with occasional application to other regions where Ashkenazi Jews lived.

THIS DATA BASE IS TEMPORARILY NOT AVAILABLE.
