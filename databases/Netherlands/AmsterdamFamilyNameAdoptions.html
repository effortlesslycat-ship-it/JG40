Amsterdam Family Name Adoptions

This database contains an index to 10,714 Amsterdam Family Name Adoption notices.

The information was compiled by Dave Verdooner and Harmen Snel and made available by Akevoth via dutchjewry.org, added to JewishGen to improve searchability features. For additional information, click here.

Most of the records include the following fields:

  • Name Status
  • Source
  • Date
  • First Name
  • Patronym
  • New Family Name
  • Address
  • Age
  • Son
  • Daughter
  • Grandson
  • Granddaughter
  • Widow
  • Child of
  • Father
  • Last Name

For further information, contact: E Randol Schoenberg.
