CryptoJews: List of those Sentenced in the Inquisition from the town of Covilha, Portugal

Background

This is a very interesting collection compiled for just that village. While the Description are all in Portuguese, if you are fortunate you can find a name, age, date and family members. The most important is the Number of the Inquisition Process or Judgement which will look something like this:

PT-TT-TSO/IL/28/3884   

With that, you can go on the website of Torre Do Tombo and see more information. This list may not be complete.

Esta es una colección muy interesante recopilada solo para ese pueblo. Si bien la descripción está en Portugués, si tiene suerte, puede encontrar el nombre, edad, fecha y miembros de una misma familia.

Lo  más importante es el número del proceso de inquisición o juicio, que se verá así:

PT-TT-TSO / IL / 28/3884

Con ese numero, puede ir al sitio web de Torre Do Tombo y ver más información. Esta lista puede ser no esta completa.

Database

This database contains records. Most of the records include the following fields:

  • First Name
  • Last Name
  • Alias
  • Year
  • Origin
  • Residence
  • Occupation
  • Sentence
  • Reference
  • Comments

For further information, contact: Genie Milgrom, Director of Crypto-Jewish Research.
