CryptoJews: Canary Island Inquisition

Background

Canary Islands
This Collection features some of the names and information that was available for the
Canary Islands Inquisition and while it is not complete it will list some genealogical
information as well and more important, the origin of the person. This is important as we
trace these genealogies and the subsequent diaspora, it is very useful to see where the
person came from to continue tracing back.

Islas Canarias
Esta colección presenta algunos de los nombres e información que estaban disponibles
para la Inquisición de las Islas Canarias y aunque no está completa, también
enumerará algunos datos genealógicos y muy importante, el origen de la persona. Esto
es muy útil ya que, si podemos rastrear estas desde su origen es posible llegar mucho
mas atrás.

Database

This database contains an index of 251 people ....Most of the records include the following fields:

  • First Name
  • Last Name
  • Alias
  • Year Arrested
  • Origin
  • Residence
  • Occupation
  • Sentence
  • Comments
  • Reference

For further information, contact: Genie Milgrom, Director of Crypto-Jewish Research.
