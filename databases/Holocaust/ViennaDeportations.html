DATABASE

This database consists of 32,325 names of individuals that were deported from Vienna, Austria.

The fields in the database are:

  • Surname
  • Given name
  • Date of birth
  • Source collection name
  • Collection ID (Arolsen Archives number)
  • Document ID#
  • Digitized source document
