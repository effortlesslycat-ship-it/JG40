<A NAME="Updates"></A>
<H3>Updates to the Database</H3>

<P>If your town is not in the above list, please consider creating 
an index of the people mentioned in your ancestral town's Yizkor Book, 
and submitting it to the Yizkor Book Project so that it can then 
be added to this ongoing effort.&nbsp;  See the JewishGen Yizkor Book 
Project's <A HREF="/Yizkor/donation/">donation procedures</A>.
</P>

<P>This database was created with the assistance of 
<A HREF="/JewishGen/Contact.asp?to=495059LR">Linda Richman</A>, 
<!-- <A HREF="/JewishGen/Contact.asp?to=58428OH">Osnat Hazan</A>, -->
the YBMNI Project Manager.&nbsp;  The previous Project Manager was 
Osnat Hazan.&nbsp;  The YBMNI Project manager coordinates the
extraction of the names from individual Yizkor Book translations,
and inputs them into Excel spreadsheets, which are then imported 
into this database.
</P>

<P>The visible fields in the database are:

<UL>
   <LI>Surname</LI>
   <LI>Given Name(s)</LI>
   <LI>Town Name, which is linked to the source,
       the original Yizkor Book where this entry appears</LI>
   <LI>Page Number(s)</LI>
   <LI>Notes / Comments</LI>
</UL>
</P>

<P>As approximately 100 new Yizkor Book translations are added to the
JewishGen Yizkor Book Project each year, this database will be added to
over time.&nbsp;  There are currently over 800 individual translations
on the <A HREF="/Yizkor/translations.html">web site</A>.
</P>


<A NAME="Sources"></A>
<H3>Finding sources</H3>

<P>To find the original source, click on the name of the town in the 
the displayed table of matching names.&nbsp;  
This will lead you to a bibliographic citation of the source 
Yizkor Book which contains that name.&nbsp;
On that page, which contains full information about that Yizkor Book,
there is a field labelled "Translation", which contains a clickable 
URL to the translated portions of that Yizkor Book on the 
JewishGen web site.
</P>

<P>For more information about Yizkor Books, see the
<A HREF="/Yizkor/">JewishGen Yizkor Book Project</A>.
</P>
