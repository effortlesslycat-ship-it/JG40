Prager Tagblatt Obituary Notices: 1877-1938

Background

This database includes 32,246 Obituary notices from the Prager Tagblatt newspaper (published in Prague) between 1877-1938.

For further information, please contact: E. Randol Schoenberg.

Searching the Database

This database can be searched via the JewishGen Austria-Czech Database or the JewishGen Unified Search
