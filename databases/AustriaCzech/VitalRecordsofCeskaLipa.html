Česká Lípa, Bohemia Vital Records

This database contains an index of names for 3 Badatelna "vital records" registers for the Czech town of Ceska Lipa (Bohmisch Leipa). The registers indexed are as follows:

  • The birth register CESKA LIPA 236 N 1797-1838 (i)
  • The marriage register CESKA LÍPA 242 O 1840-1907 (i)
  • The marriage register CESKA LÍPA 243 O 1907-1949 (i)

Each of these three registers already contains a handwritten index, but they have shortcomings. The birth register index is alphabetized by given name, making it difficult to search for a particular surname. The marriage registers index only the male names, so half the names that might be of interest are missing.
