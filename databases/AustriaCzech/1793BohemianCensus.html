1793 Bohemian Census

In Bohemia, the western half of the Czech Republic, a census of Jewish families was conducted in 1793. For Prague, there was a census in 1794. The census covers the following districts (Kreis):

  1. BEROUNSKY ( Beroun )
  2. BOLESLAVSKY ( Mlada Boleslav / Jung Bunzlau )
  3. BUDEJOVICKY ( Ceske Budejovice / Budweis )
  4. BYDZOVSKY ( Novy Bydzov / Neu Bidschow)
  5. CASLAVSKY ( Caslav )
  6. HRADECKY ( Hradec Kralove / Königgratz )
  7. CHRUDIMSKY ( Chrudim )
  8. KLATOVSKY (Klatovy / Klattau )
  9. KOURIMSKY ( Kourim )
  10. LITOMERICKY ( Litomerice / Leitmeritz )
  11. LOKETSKY (Loket / Elbogen )
  12. PLZENSKY ( Plzen / Pilsen )
  13. PRACHENSKY ( Prachatice / Prachatitz )
  14. RAKOVNICKY ( Rakovnik )
  15. TABORSKY ( Tabor )
  16. ZATECKY ( Zatec / Saaz )
  17. PRAHA

A team led by Prof. Ivana Ebelova at the Charles University in Prague transcribed the census and published them. The books can also be downloaded from http://pvh.ff.cuni.cz/soupisy.htm. JewishGen has converted the data for inclusion in its Austria-Czech database.
