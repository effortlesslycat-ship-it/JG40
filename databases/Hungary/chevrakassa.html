Kassa Chevra Kadisha Members Registration Book
Data compiled and transcribed by Peter Absolon

  

This database contains 825 Jewish names of Chevra Kadisha members from Kosice (Kassa) Jewish Community, recorded in years 1908-1924 and listing members since 1859.

Introduction

The registration book of Kassai Chevra Kadisha is property of Kehila Kosice, Jewish Community of Kosice, Slovakia. Every page contains names of three individuals that were members of Neolog Chevra Kadisha. I have personally scanned all the pages and transcribed all the names into the format suitable for uploading the data into JewishGen All Hungary Database.

The Database

The database includes the following fields:  

  • Page number / Record number

  • Surname

  • Birt surname (if changed later)

  • Given name

  • Hebrew name

  • Father's Hebrew name

  • Registration date

  • Place of origin

Although the records don’t contain much genealogical data, in most of the cases they mention Hebrew name of the Chevra Kadisha member as well as his father’s Hebrew name, which can be useful in genealogical research.
